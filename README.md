# Portfolio

My personal website portfolio which is currently being hosted on Cloudflare Pages.

This project introduces me and the work I've done.  It also shows the tech stack I currently use/have used in the past projects.

Why this design and layout?
- I wanted to keep it simple and straight to the point. I initially want it to be similar to a social media UI where you have your profile, the posts, friends, etc...
- I somewhat achieved that. Instead of posts, you can see the projects I have done. The latest one being at the center, the previous ones being at the side.

[Tech Stack]
- Next.js, React.js, Vite, TailwindCSS
- Supabase Storage

[Deployment]
- Cloudflare Pages
